## GitHub Beyond Source Control

GitHub is not just a code hosting platform. It's a **complete developer platform** that combines:

* **Source Control**: Git repository hosting
* **Collaboration**: Issues, PRs, Discussions, Reviews
* **Automation**: GitHub Actions, CI/CD workflows
* **Security**: Code scanning, dependency management, secrets
* **Governance**: Access control, compliance, auditing
* **Project Management**: Projects, Boards, Milestones

---

## GitHub's Product Ecosystem

### Core Products

* **GitHub.com**: Main platform (cloud-hosted)
* **GitHub Enterprise Server**: Self-hosted solution
* **GitHub Enterprise Cloud**: Managed enterprise solution
* **GitHub Actions**: CI/CD automation
* **GitHub Packages**: Package registry
* **GitHub Pages**: Static site hosting
* **GitHub Codespaces**: Cloud development environments

### Supporting Tools

* **GitHub Desktop**: Desktop application
* **GitHub Mobile**: Mobile application
* **GitHub CLI**: Command-line interface
* **GitHub API**: Programmatic access

## Platform Architecture

### Account Hierarchy

```
Enterprise Account
    │
    ├─── Organization 1
    │    ├─── Repository 1
    │    └─── Repository 2
    │
    └─── Organization 2
         ├─── Repository 3
         └─── Repository 4
```

### Key Concepts

* **Account**: Identity (personal or organization)
* **Organization**: Ownership and billing boundary
* **Repository**: Code container
* **Plan**: Feature entitlement
