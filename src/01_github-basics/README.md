# GitHub Basics

This folder covers **Domain 1: GitHub Fundamentals (20%)** of the GitHub Foundations Certification, including foundational concepts about Git, GitHub, and the platform itself.

---

## Domain 1: GitHub Fundamentals (20%)

**Topics Covered**:

* Version control basics and Git fundamentals
* GitHub platform overview
* GitHub accounts, plans, and billing
* GitHub Desktop and GitHub Mobile
* Repository basics and GitHub Flow

---

## Files

### Core Concepts

* `01_version-control-basics.md` - What is version control and why it matters
* `02_git-essentials.md` - Git commands and workflows
* `03_github-essentials.md` - GitHub platform overview and account setup
* `04_github-flow.md` - Standard GitHub workflow: clone, branch, push, merge

### Platform Fundamentals

* `05_github-platform-overview.md` - GitHub as a product platform
* `06_accounts-and-plans.md` - Personal, Organization, and Enterprise accounts
* `07_billing-and-usage.md` - Billing, license usage, and metered features
* `08_github-desktop-mobile.md` - GitHub Desktop and GitHub Mobile applications

---

## Learning Path

1. Start with `01_version-control-basics.md` to understand why version control exists
2. Study `02_git-essentials.md` for Git command fundamentals
3. Review `03_github-essentials.md` for GitHub platform basics
4. Practice `04_github-flow.md` for the standard workflow
5. Understand `05_github-platform-overview.md` for platform awareness
6. Learn `06_accounts-and-plans.md` for account and plan types
7. Study `07_billing-and-usage.md` for billing and usage concepts
8. Explore `08_github-desktop-mobile.md` for alternative access methods
