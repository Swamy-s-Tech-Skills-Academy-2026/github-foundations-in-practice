# Collaboration Features

This folder covers **Domain 3: Collaboration Features (30%)** and **Domain 5: Project Management (7%)** of the GitHub Foundations Certification.

---

## Domain 3: Collaboration Features (30%)

**Topics Covered**:

- Pull requests and code reviews
- Issues and discussions
- CODEOWNERS and review requirements
- Collaborative workflows

**Files**:

- `pull-requests.md` - PR workflow and best practices
- `code-reviews.md` - Review process and requirements
- `issues-projects.md` - Issues, Projects, Boards, and Milestones
- `discussions.md` - Community discussions

---

## Domain 5: Project Management (7%)

**Topics Covered** (included in `issues-projects.md`):

- GitHub Projects and Boards
- Milestones
- Project automation
- Project templates

**Note**: Project Management content is integrated into the collaboration folder since Projects and Boards are collaboration tools.

---

## Learning Path

1. Start with `pull-requests.md` to understand the PR workflow
2. Review `code-reviews.md` for review processes
3. Study `issues-projects.md` for both Issues and Project Management
4. Explore `discussions.md` for community engagement
