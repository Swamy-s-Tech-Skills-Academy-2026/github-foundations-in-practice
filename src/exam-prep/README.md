# Exam Preparation Materials

This folder contains materials specifically focused on GitHub Foundations Certification exam preparation.

---

## Contents

### 01_key-definitions.md

Essential definitions and terminology for the exam. Key concepts that frequently appear in exam questions.

### 02_trick-questions.md

Common trick questions and ambiguity traps in the exam. Questions designed to test deep understanding rather than surface knowledge.

### 03_exam-mindset.md

Strategies for approaching the exam, time management, and mindset tips for success.

---

## How to Use

1. **Review key definitions** before starting each domain
2. **Study trick questions** after completing all domains
3. **Read exam mindset** before taking the exam
