# Exam Readiness Checklist

Use this checklist to self-assess readiness for the GitHub Foundations exam.

## Domain Coverage

- [ ] GitHub Basics (concepts + key terms)
- [ ] Working with Repositories (clone, commit, branches, merge/rebase)
- [ ] Collaboration (PRs, reviews, issues, discussions)
- [ ] GitHub Tools (Actions, Pages, Marketplace)
- [ ] Security & Best Practices (permissions, branch protection, secrets)
- [ ] Open Source & Community (licenses, contributing, code of conduct)

## Practice & Review

- [ ] Completed labs in `src/labs/`
- [ ] Reviewed common mistakes in `src/notes/common-mistakes.md`
- [ ] Reviewed glossary in `src/notes/glossary.md`
- [ ] Completed practice questions in `src/resources/practice-questions.md`
- [ ] Revisited weak domains and updated notes

## Final Checks

- [ ] Can explain Git vs GitHub and core terminology
- [ ] Comfortable with basic Git commands and typical workflows
- [ ] Understand PR lifecycle and code review basics
- [ ] Know where to configure and run GitHub Actions workflows
- [ ] Know common security controls (secrets, protection rules, permissions)
