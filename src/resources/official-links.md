### Official Certification Page

**Primary Resource**: [GitHub Foundations Certification - Microsoft Learn](https://learn.microsoft.com/en-us/credentials/certifications/github-foundations/?practice-assessment-type=certification)

This is the official page for the GitHub Foundations certification (Exam GH-900) where you can:

- View exam overview and requirements
- Access practice assessments
- Schedule your exam through Pearson VUE
- Download the exam study guide
- Access exam sandbox for practice

### Key Information

- **Exam Duration**: 100 minutes
- **Certification Validity**: 2 years
- **Level**: Beginner
- **Exam Provider**: Microsoft (maintained by GitHub)
- **Languages**: English, Spanish, Portuguese (Brazil), Korean, Japanese

### Exam Domains & Weighting

According to the official exam guide:

- **Domain 1**: Introduction to Git and GitHub - 22%
- **Domain 2**: Working with GitHub Repositories - 8%
- **Domain 3**: Collaboration Features - 30%
- **Domain 4**: Modern Development - 13%
- **Domain 5**: Project Management - 7%
- **Domain 6**: Privacy, Security, and Administration - 10%
- **Domain 7**: Benefits of the GitHub Community - 10%

### Practice Resources

- **Practice Assessment**: Available on the official certification page
- **Exam Sandbox**: Experience the exam interface before taking the actual exam
- **Study Guide**: Download the official GH-900 study guide

### Exam Registration

- Schedule through [Pearson VUE](https://www.pearsonvue.com/)
- **Important**: Register with a personal MSA account (not organizational AAD account) to avoid losing exam records
- **Retake Policy**: 24 hours after first attempt, varying times for subsequent retakes
